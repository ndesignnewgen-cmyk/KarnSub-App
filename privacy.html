1. ຂໍ້ມູນທີ່ເຮົາເກັບ

ເມື່ອເຈົ້າເຂົ້າສູ່ລະບົບດ້ວຍ Google (ສຳລັບ PRO) ເຮົາເກັບ: ອີເມວ ແລະ ຊື່ບັນຊີ — ໃຊ້ເພື່ອຢືນຢັນສະຖານະ PRO ເທົ່ານັ້ນ.

2. Gemini API Key (BYOK)

KarnSub ໃຊ້ Google Gemini AI ສຳລັບຖອດສຽງ, ພາກສຽງ ແລະ ກວດ slip. ເຈົ້າໃຊ້ API key ຂອງເຈົ້າເອງ. key ນີ້ເກັບໄວ້ໃນເຄື່ອງຂອງເຈົ້າເທົ່ານັ້ນ ບໍ່ໄດ້ສົ່ງໄປ server ຂອງເຮົາ.

3. ການປະມວນຜົນສື່

ໄຟລ໌ສຽງ/ວິດີໂອ/ຮູບ ທີ່ເຈົ້າປະມວນຜົນ ຈະຖືກສົ່ງໄປຫາ Google Gemini ໂດຍກົງ (ຜ່ານ key ຂອງເຈົ້າ) ເພື່ອປະມວນຜົນ. ເຮົາບໍ່ໄດ້ເກັບສຳເນົາໄຟລ໌ເຫຼົ່ານີ້.

4. ການຈ່າຍເງິນ

ເມື່ອສະໝັກ PRO ເຈົ້າອັບໂຫຼດ slip ການໂອນ. ເຮົາເກັບເລກອ້າງອີງ (ref) ການໂອນ ເພື່ອກັນການໃຊ້ຊ້ຳ. ບໍ່ເກັບເລກບັນຊີເຕັມຂອງເຈົ້າ.

5. ການແບ່ງປັນກັບພາກສ່ວນທີ່ສາມ

ເຮົາໃຊ້ Google Firebase (Auth/Firestore) ແລະ Google Gemini. ເຮົາບໍ່ຂາຍຂໍ້ມູນຂອງເຈົ້າໃຫ້ໃຜ.
