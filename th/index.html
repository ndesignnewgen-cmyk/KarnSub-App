การติดตั้ง

เริ่มใช้ใน 3 ขั้นตอน

เนื่องจากไม่ได้อยู่บน Play Store — ต้องเปิด "ติดตั้งจากแหล่งที่ไม่รู้จัก" ก่อน

1

ดาวน์โหลด APK

กด "ดาวน์โหลด APK" ด้านล่าง → รอโหลดให้เสร็จ

2

อนุญาตการติดตั้ง

เปิดไฟล์ → ถ้ามีคำเตือน กด "การตั้งค่า" → เปิด "อนุญาตจากแหล่งนี้"

3

ใส่ key → ใช้ได้เลย

เปิดแอป → ใส่ Gemini API key (ฟรี) → เริ่มทำซับได้ทันที

FAQ

ถอดเสียงเป็นซับภาษาไทยได้ไหม?

ได้ — KarnSub ถอดเสียงภาษาไทยเป็นซับไตเติลอัตโนมัติด้วย AI พร้อมฟอนต์ไทย และแปลไทย↔ลาว↔อังกฤษ รวมถึง UI ภาษาไทย (สลับได้ในหน้าตั้งค่า)

Gemini API key คืออะไร หาได้ที่ไหน?

KarnSub ใช้ AI ของ Google (Gemini) คุณต้องมี key ของตัวเอง (ฟรี) จาก aistudio.google.com/apikey แล้วนำมาใส่ในแอป โดย key จะเก็บไว้ในเครื่องคุณเท่านั้น

ทำไมไม่มีบน Play Store?

แจกตรงผ่านเว็บไซต์นี้เพื่อให้สมัคร PRO ผ่าน QR ได้สะดวก ไฟล์ APK เซ็น (sign) เรียบร้อย ปลอดภัย

ใช้กับมือถือรุ่นไหนได้บ้าง?

Android 7.0 ขึ้นไป

ข้อมูลปลอดภัยไหม?

วิดีโอ/เสียงประมวลผ่าน Gemini (Google) ด้วย key ของคุณเอง เราเก็บเพียงอีเมล (สำหรับ login) ไม่ขายข้อมูล
