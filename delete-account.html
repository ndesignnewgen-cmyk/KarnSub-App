ວິທີລຶບບັນຊີ ແລະ ຂໍ້ມູນ

ອັບເດດ: 5 ມິຖຸນາ 2026 · ແອັບ: KarnSub (com.anniekaydee.subtitle_app)

ວິທີລຶບໃນແອັບ (ງ່າຍສຸດ)

  1. ເປີດແອັບ KarnSub
  2. ໄປທີ່ ຕັ້ງຄ່າ (Settings)
  3. ໃນສ່ວນ ບັນຊີ (ຕ້ອງເຂົ້າສູ່ລະບົບກ່ອນ) ກົດ “ລຶບບັນຊີ”
  4. ຢືນຢັນ → ບັນຊີ ແລະ ຂໍ້ມູນ PRO ຈະຖືກລຶບຖາວອນ

ຖ້າເຂົ້າແອັບບໍ່ໄດ້

ສົ່ງອີເມວຫາ nouchthv@gmail.com ດ້ວຍຫົວຂໍ້ “ລຶບບັນຊີ KarnSub” ພ້ອມບອກອີເມວ Google ທີ່ໃຊ້ເຂົ້າສູ່ລະບົບ. ເຮົາຈະລຶບໃຫ້ພາຍໃນ 7 ມື້.

ຂໍ້ມູນທີ່ຈະຖືກລຶບ

  • ບັນຊີ Google ທີ່ເຊື່ອມຕໍ່ (Firebase Auth)
  • ອີເມວ, ຊື່ບັນຊີ, ສະຖານະ/ວັນໝົດອາຍຸ PRO

ໝາຍເຫດ: ໂປຣເຈັກ, ວິດີໂອ, subtitle ແລະ API key ຂອງເຈົ້າ ເກັບຢູ່ໃນເຄື່ອງເທົ່ານັ້ນ — ການຖອນຕິດຕັ້ງແອັບ ຈະລຶບຂໍ້ມູນເຫຼົ່ານີ້ໝົດ.

ຂໍ້ມູນທີ່ເກັບໄວ້ (ຕາມກົດໝາຍ)

ບັນທຶກ ref ການຈ່າຍເງິນ (ສຳລັບ accounting/ກັນໃຊ້ຊ້ຳ) ອາດເກັບໄວ້ສູງສຸດ 1 ປີ ໂດຍບໍ່ຜູກກັບຕົວຕົນ.

วิธีลบบัญชีและข้อมูล

อัปเดต: 5 มิถุนายน 2026 · แอป: KarnSub (com.anniekaydee.subtitle_app)

วิธีลบในแอป (ง่ายสุด)

  1. เปิดแอป KarnSub
  2. ไปที่ ตั้งค่า (Settings)
  3. ในส่วน บัญชี (ต้องเข้าสู่ระบบก่อน) กด “ลบบัญชี”
  4. ยืนยัน → บัญชีและข้อมูล PRO จะถูกลบถาวร

ถ้าเข้าแอปไม่ได้

ส่งอีเมลถึง nouchthv@gmail.com หัวข้อ “ลบบัญชี KarnSub” พร้อมแจ้งอีเมล Google ที่ใช้เข้าสู่ระบบ เราจะลบให้ภายใน 7 วัน

ข้อมูลที่จะถูกลบ

  • บัญชี Google ที่เชื่อมต่อ (Firebase Auth)
  • อีเมล, ชื่อบัญชี, สถานะ/วันหมดอายุ PRO

หมายเหตุ: โปรเจกต์ วิดีโอ ซับไตเติล และ API key ของคุณ เก็บอยู่ในเครื่องเท่านั้น — การถอนการติดตั้งแอปจะลบข้อมูลเหล่านี้ทั้งหมด

ข้อมูลที่เก็บไว้ (ตามกฎหมาย)

บันทึก ref การชำระเงิน (สำหรับบัญชี/กันใช้ซ้ำ) อาจเก็บไว้สูงสุด 1 ปี โดยไม่ผูกกับตัวตน
